# Salary Calculator

A simple web-based Salary Calculator that helps users calculate their salary based on input values.

## Features
- Input monthly Net salary
- Calculate salary instantly
- Simple and user-friendly interface
- Works directly in the browser
- Lightweight JavaScript project

## How It Works
1. Enter your Net salary amount.
2. Click the **Calculate** button.
3. The system will instantly display the result.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Download or clone this repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.
